Rationale

The brief: make merch that developers actually want to wear. Not branded giveaways — real gear with personality. The solution was a system of five original retro-cartoon characters, each carrying a witty dev-culture phrase. Figures rooted in nostalgia (Eat, Sleep, Deploy; Cloud State of Mind; Clean Code Club) applied consistently across a full product range — resonating with the community without ever feeling like an advertisement.
